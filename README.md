## General Infomation 
This project is used to collect data set from the 4D Website. Data collected are: 

* First prize
* Second prize
* Third prize
* Starter prize
* Consolation prize

## Technology/Features
Project is created with:
* BeautifulSoup(bs4)
* Python FileWriter
* Regex(Regular Expression)
